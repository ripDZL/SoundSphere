# Build backups

Pristine snapshots of the last stable builds, kept for quick rollback.

- `Chrome-1.1/SoundSphere/`  — original Chrome 1.1 build
- `FireFox-1.2/SoundSphere/` — original Firefox 1.2 build

These folders are NOT packaged by CI. To roll back, copy the relevant
SoundSphere folder from here over `Chrome/1.1` or `FireFox/1.2`.
